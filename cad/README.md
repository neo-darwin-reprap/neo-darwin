# Neo-Darwin CAD Directory

This directory contains all CAD-related files for the Neo-Darwin 3D printer project.

## Quick Start

### Option 1: Full Wizard (Recommended for New Users)

```bash
./setup.sh
```

The setup wizard will:
1. Check Python installation
2. Create a virtual environment
3. Install required packages
4. Run the configuration wizard
5. Build all parts

### Option 2: Self-Setup + Build

```bash
# Create and activate virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run configuration wizard
./configure.py

# Build all parts
./build.sh build_all
```

## Scripts

### setup.sh
Full setup wizard for new users. Handles environment setup, configuration, and building.

### configure.py
Interactive configuration wizard. Run this to customize:
- Build volume (X, Y, Z)
- Frame dimensions
- Extruder type (Wade)
- Hotend type
- Electronics (mainboard, Z-probe, CANbus)

### build.sh
Build script to generate STL files from Python CAD scripts.

**Usage:**
```bash
./build.sh build_all          # Build all parts
./build.sh build corner_front_left  # Build specific part
./build.sh list              # List all available parts
./build.sh clean             # Delete all STLs
./build.sh help              # Show help
```

## Directory Structure

```
cad/
├── parts/          # Python CAD scripts for each part
├── stl/           # Generated STL files (output directory)
├── config.py       # Configuration file (generated by configure.py)
├── requirements.txt   # Python dependencies
├── setup.sh        # Full setup wizard
├── configure.py    # Configuration wizard
└── build.sh       # Build script
```

## Parts

Currently available parts:
- `corner_front_left` - Front-left corner with integrated Z1 motor
- `corner_front_right` - Front-right corner with integrated Z2 motor

More parts will be added as the project progresses.

## Configuration

Edit `config.py` directly or run `./configure.py` for an interactive wizard.

Key configuration parameters:
- `BUILD_VOLUME` - X, Y, Z dimensions
- `M12_FIT_DIA` - M12 rod diameter with lumpy factor
- `EXTRUDER_TYPE` - "WADE" or other
- `TRIPLE_Z` - Enable/disable triple Z system
- `Z_MOUNT_PATTERN` - Z-motor mounting configuration

## Dependencies

- Python 3.9+
- build123d
- numpy

See `requirements.txt` for specific versions.
